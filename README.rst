=======
Cipr UI
=======

User interface utilities for Corona

Installation
============

Installation is done with `cipr <http://github.com/six8/corona-cipr>`_

::

    cipr install:git://github.com/six8/cipr.ui.git

Usage
=====

::

    local cipr = require 'cipr'
    local ui = cipr.import('cipr.ui')
